Sistema Central : Nome genérico das aplicações que integram as APIs dos parques. São clientes do parque.

API.Parque : Refere-se à API individual de cada parque público ou privado. Devolve uma lista de lugares de estacionamento disponíveis para aluguer, e permite efectuar reservas.

Utilizador não registado: É o programador de um sistema central que pretende integrar a API no seu sistema e necessita de se registar na API para obter credenciais para a utilizar.

api/reservasController : É o Controller da API responsável pelo CRUD das reservas.

api/reserva : É o Model da reserva na API.

context : É o nome da base de dados da API.

api/usersController : É o Controller da API responsável por registar e fazer login de utilizadores.

api/user : É o Model do user na API.
